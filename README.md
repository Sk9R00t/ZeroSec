# Discord Moderation Bot with Web Dashboard

## Overview
This project is a powerful Discord moderation bot designed to manage and enhance Discord communities. It features a web-based dashboard that allows administrators to configure settings, monitor activity, and gain insights into server performance.

## Features
- **User Management**: Kick, ban, mute, or warn users with ease.
- **Real-time Logging**: Monitor member actions and bot activity in real-time.
- **Web Dashboard**: A user-friendly interface to control the bot and view analytics.
- **Custom Commands**: Create and manage custom commands for user interaction.
- **Role Assignment**: Automatically assign roles based on predetermined criteria.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Sk9R00t/ZeroSec.git
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```
3. Configure the bot token and other settings in the `.env` file.
4. Run the bot:
   ```bash
   npm start
   ```

## Usage
- Invite the bot to your server using the generated link.
- Access the web dashboard at `http://localhost:3000` to configure settings.
